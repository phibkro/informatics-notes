Questions

  1. The server serves and sends the client requested information if the client has the required access
  2. The client requests information from the server
  3. An IP(Internet Protocol) address is a unique identifier and destination assigned to computers
  4. DNS stands for Domain Name System and it effectively is a phonebook (which can be thought of like a key:value database) connecting human readable Domain Names like "www.google.com" to unreadable IP addresses like "192.0.2.1"
  5. The Internet refers to the system of interconnected computers that utilizes TCP/IP to communicate between devices and networks. WWW(World Wide Web) refers to the Web which is an information system on top of the internet that lets web resources be accessed over the Internet using HTTP.
  6. HTML stands for HyperText Markup Language and it is a text format used by the web for documents meant to be displayed in a web browser
  7. CSS files are kept inside the head element and not the body since CSS is styling information about the body, not the body itself, which contain the structure and content of the web page
  8. Inline styles override rules defined in style elements and external stylesheets because inline styles are of a higher specificity than the previously mentioned methods. In addition, inline styles are computed last and the lastly computed code takes precedence over earlier code
